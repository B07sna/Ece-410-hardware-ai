## Tentative Project Topic

**Workload Analysis and Hardware Efficiency of Quantized MobileNetV2**

This project analyzes and compares the computational workload (MACs, memory usage, arithmetic intensity) of MobileNetV2 running at full precision (FP32) versus INT8 quantization, with the goal of understanding hardware efficiency tradeoffs on edge and mobile devices.
